HNTech: Incredible Software For Windows Machines

Made with Python, Open Source

HNTech is a project I initiated to build software for Windows machines. My apps are built with Python and Tkinter and open source. My main goal is to provide a suite of helpful tools for anyone who wishes to do anything. That's why most of my software is designed for beginners and not professionals. My apps are completely free from malware and I assure you they are not destructive in any way. Source code for my apps is available on Github.
